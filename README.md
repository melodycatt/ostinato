# ostinato, a higher-level graphics library built on wgpu
ostinato is designed as a graphics library that is more accessible, eventually offering all the functionality of wgpu for projects that need specific features, while removing all the boilerplate and confusing code for simpler projects or beginners. it was born out of me learning wgpu and seeing how quickly the project bloated with what felt like reinventing the wheel over and over. to remedy this, ostinato both provides overridable default values for almost everything, as well as prebuilt structs or traits for common things like meshes and vertex buffers and premade mateirals like blinn-phong shading.

## performance
as this is a wrapper on top of wgpu, there will always be _some_ overhead, however i try keep it to a mininum. keep in mind this is my first library so there are probably countless little performance mistakes - this is why its open source!
